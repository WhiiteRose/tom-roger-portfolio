## ✦ Overview

One-page portfolio with polished animations — no bundler, no build step.  
Animated loader, custom cursor, scroll reveals, parallax, smooth transitions.  
Dark editorial design with a red accent, built entirely from scratch.

---

## 🛠 Stack

| Layer | Technology |
|---|---|
| UI | React 18 (UMD / CDN) |
| Style | TailwindCSS (CDN) + custom CSS |
| Animation | GSAP 3 · ScrollTrigger · ScrollToPlugin |
| Typography | Inter Tight · Instrument Serif · JetBrains Mono |

---

## 📁 Structure

```
├── index.html              # Entry point
├── portfolio-app.jsx       # Main sections (Hero, Work, About, Stack, Contact)
├── portfolio-sections.jsx  # Shared components (Nav, Footer, Cursor, Loader…)
├── portfolio.css           # Base styles & GSAP helpers
├── CV.html                 # Online résumé
├── favicon.png             # Favicon 512px
├── favicon-64.png          # Favicon 64px
├── apple-touch-icon.png    # iOS icon
├── images/                 # Static assets
└── uploads/                # Project media
```

---

## 🚀 Run locally

```bash
# With npx serve (recommended)
npx serve .

# Or Python
python -m http.server 8080
```

> ⚠️ Don't open `index.html` directly via `file://` — Babel standalone can't load the JSX files that way.
